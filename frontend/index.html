Willkommen bei fordify
Ihre Daten bleiben bei Ihnen

fordify speichert alle Daten ausschließlich in Ihrem Browser (localStorage). Kein Mandantendatum verlässt Ihr Gerät – kein Server, keine Cloud, DSGVO-konform.

Wichtig: Datensicherung

Browser können gespeicherte Daten bei Cache-Bereinigung oder einem Gerätewechsel verlieren. Exportieren Sie Ihre Fälle regelmäßig als JSON-Datei (Fälle → Export JSON) und bewahren Sie die Datei sicher auf.

Geräteübergreifend nutzen

Den JSON-Export können Sie auf einem anderen Gerät oder Browser über Fälle → Import JSON wieder einlesen.
